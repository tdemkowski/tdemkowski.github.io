Online Store

View App

Source code

github.com/tdemkowski/Online-Store

An Online Store where clients can create an create an account and purchase/sell products. It uses EJS,CSS, NodeJs, ExpressJs and Mongoose (MongoDB). Stripe API is used to deal with transactions.

Using Mailgun, clients who sign-up will have a confirmation email sent to them by the Store. Users who want their password reset are emailed a link with a generated token (using the Json Web Token library) giving them access to changing their password.

There is user authentication (such as CSRF tokens) to give access or restrict users from interacting with what they should or should not be able to. I use express-validator to handle all routes. Sinon and Chain were used for unit testing the backend controllers.

Instagram Clone

Source code

https://github.com/tdemkowski/Instagram-Clone

Massive project replicating Instagram that is still in development, with the goal of having over 90% of the original site's functionality. Collaborated with one other developer. It runs on TypeScript, React, Node, Express, and PostgreSQL.

I have built most of the frontend with React from scratch, with some help of a React UI library Ant Design. Clients have the ability to sign-up and log-in to their account, create posts and set their own profile details.

We have backend user/client authentication that we built entirely from scratch (with the exception of using Json Web Tokens) using OOP. The foundation of this was set by my collaborator, where I focused on wiring it up to the frontend and managing bugs. The database is currently generated with users and posts via the typeorm-seeding library, filled with properties including email addresses, account names, comments, posted-images, etc.
